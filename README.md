# Linux Toolkit

Successor to [auto-secure-linux](https://github.com/d3vop-llc/auto-secure-linux).

## Install

**If `curl` doesn't work, then change `v0.0.8` to current highest (or preferred) version.**

### Seperate Commands

Download binary

```bash
sudo curl -L "https://github.com/d3vop-llc/linux-toolkit/releases/download/v0.0.8/linux-toolkit" -o /usr/bin/linux-toolkit
```

Allow binary to be executed

```bash
sudo chmod +x /usr/bin/linux-toolkit
```

Run binary

```bash
linux-toolkit
```

### All in one command

```bash
sudo curl -L "https://github.com/d3vop-llc/linux-toolkit/releases/download/v0.0.8/linux-toolkit" -o /usr/bin/linux-toolkit
sudo chmod +x /usr/bin/linux-toolkit
linux-toolkit
```
